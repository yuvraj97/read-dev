Central Limit Theorem says that: \(\left| \overline{X}_n-\mu \right| \leq \frac{C\sigma}{\sqrt{n}}\quad;\) \(C\) is a constant and choice of \(C\) depends on us.
But it's true only if sample size \(n\) is large enough, So what if the sample size is not large enough?
That's where Hoeffding's inequality kicks in.


Hoeffding's inequality

Say that we have \(n\) i.i.d. random variables, \(X_1,X_2,....,X_n \stackrel{iid}{\sim } X\)
such that \(\mathbb{E}[X]=\mu\) and
\(X \in [a,b]\) almost surely
  • Here we need our random variables to be bounded, so it rules out Gaussian, exponential, everything that goes to \(\infty\) or \(-\infty\) are ruled out
    examples for bounded random variables are \(\mathcal{U}nif[0,1], Ber(0.5),....\)
    \(\mathcal{U}nif[0,1]\) is a Uniform random variable that takes values between \(0\) and \(1\)
    \(Ber(0.5)\) is a Bernoulli random variable that takes values either \(0\) or \(1\)
Then according to Hoeffding's inequality:
\[\mathbb{P}\left(\left|\overline{X}_n -\mathbb{E}[X]\right| \geq \epsilon \right) \leq 2 \exp\left(-\frac{2n\epsilon ^2}{(b-a)^2}\right)\quad \forall\epsilon >0\]
If we do satisfy the conditions described above, then according to Hoeffding's inequality probability of deviation of \(\overline{X}_n\) around \(\mu\) being greater then \(\epsilon\) decays exponentially w.r.t. \(n\).
Well, what's nice about this is that \(\epsilon\) need not to be too small,
like if \(\epsilon = C/\sqrt{n}\quad ; C\) is a constant Then the probability will be: \[\mathbb{P}\left(\left|\overline{X}_n -\mathbb{E}[X]\right| \geq \frac{C}{\sqrt{n}} \right) \leq 2 e^\left(-\frac{2C^2}{(b-a)^2}\right)\] For example:
If \(X_1,X_2,....,X_n \stackrel{iid}{\sim } Ber(p);\quad\) and \(\epsilon=\frac{C}{\sqrt{n}}\)
Then: \[\mathbb{P}\left(\left|\overline{X}_n -\mathbb{E}[X]\right| \geq \frac{C}{\sqrt{n}} \right) \leq 2 e^{-2C^2}\] It's true for all \(n\), and for \(C\) large enough it behaves like a Gaussian.

Remember that Kiss Example:
 \(R_1,R_2,....,R_n \stackrel{iid}{\sim } Ber(p);\quad\)
\(\overline{R}_n = \frac{1}{n}\sum^n_{i=1}R_i\quad,\quad\) \(Var(R_i)=p(1-p)\)

  • Law of Large Numbers(LLN)

    \[\overline{R}_n \xrightarrow [n\to \infty ]{\mathbb{P},\text{ a.s.}} p\] \(\overline{R}_n\quad\) is the average of people that turn there head to the right.
    \(p\quad\) it's the actual proportion of people that turn there head to the right.
    So according to LLN as \(n\) becomes larger \(\overline{X}_n\) converges to \(p\quad\) OR (say) \(\overline{R}_n\) become a consistent estimator.
    Consistency is important .
    Now we know that \(\overline{R}_n\) is a consistent estimator, but we don't how good is it. We can check it by looking at the deviation of \(\overline{R}_n\) from \(p\). \(\quad\mathbb{P}(|\overline{R}_n -p|)=?\quad\), CLT will help us finding it.

  • Central Limit Theorem(CLT)

    \[ \sqrt{n}\frac{\overline{R}_n - p }{\sqrt{p(1-p)}} \xrightarrow [n\to \infty ]{(d)} \mathcal{N}(0,1) \\ \] \[ \overline{R}_n - p \xrightarrow [n\to \infty ]{(d)} \mathcal{N}\left(0,\frac{p(1-p)}{n}\right) \]
    if \(n\) is large enough:
    \[ \overline{R}_n - p \approx \mathcal{N}\left(0,\frac{p(1-p)}{n}\right) \\ \Rightarrow \mathbb{P}(|\overline{R}_n -p| > \epsilon) \approx \mathbb{P}\left( \left| \mathcal{N}\left( 0,\frac{p(1-p)}{n} \right) \right| \geq \epsilon \right) \]

  • Hoeffding's inequality

    \[\mathbb{P}(|\overline{R}_n -p| \geq \epsilon ) \leq 2 e^\left(-2n\epsilon^2\right)\]
And if we put \(n=124,\hat{p}=0.645\quad\) then:
In CLT:
\(\mathbb{P}(|\overline{R}_n -p| \geq 0.084 )\approx 5\%\)
And in Hoeffding's inequality:
\(\mathbb{P}(|\overline{R}_n -p| \geq \epsilon ) \leq 35\%\)

Well \(35\%\) is a pretty big number, Hoeffding's inequality says that there is \(35\%\) chance that we may deviate more than \(0.084\) is \(\leq 35\%\)
And CLT says the probability of this happening is \(\approx 5\%\)
So Hoeffding's inequality is more conservative then CLT but it's also less precise.
