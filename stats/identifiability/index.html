Previously we talked about a model being well specified or misspecified.
Now we ask is the parameter Identifiable?

The parameter \(\theta\) is identifiable iff it maps \(\theta \in \Theta \mapsto \mathbb{P}_\theta\) is injective.
By injectivity, we can say that if we have two parameters say \(\theta\) and \(\theta'\) and if \(\theta\neq \theta'\) then it implies that we have two different distributions \(\mathbb{P}_\theta\neq\mathbb{P}_{\theta'}\)
Or say that by two different parameters we cannot end up with the same distribution.
So by injectivity, we have a unique parameter for a distribution.
If \(\theta\) and \(\theta'\) give us the same distribution there is no way we can find out the correct parameter even if we have \(\infty\) amount of data.
So in order to estimate the parameter it must be identifiable.

In examples shown in previous page, parameter is identifiable.
Example:
If: \[ X_i = 𝟙_{Y_i \gt 0}\text{ (indicator function)} \\ \ \\ \text{OR} \\ \ \\ X_i = \left\{\begin{matrix} 1\text{ ; if } Y_i \gt 0 \\ 0\text{ ; if } Y_i \leq 0 \end{matrix}\right. \] and \(Y_1,\cdots,Y_n \stackrel{iid}{\sim } \mathcal{N}(\mu,\sigma^2)\) for some unknown \(\mu\in\mathbb{R}\) and \(\sigma^2\gt 0\)
And the question is can we find \(\mu\) and \(\sigma^2\)?
Solution
\[X_i \sim Ber(\mathbb{P}(Y \gt 0)) \\ \ \\ \mathbb{P}(Y \gt 0)= \mathbb{P}\left(\frac{Y - \mu}{\sigma} \gt \frac{-\mu}{\sigma}\right)\\ \Rightarrow \mathbb{P}(Y \gt 0)= 1-\mathbb{P}\left(\frac{Y - \mu}{\sigma} \leq \frac{-\mu}{\sigma}\right) \\ \text{ (by standardization)} \\ \Rightarrow \mathbb{P}(Y \gt 0)= 1-\mathbb{P}\left(Z \leq \frac{-\mu}{\sigma}\right)\\ \{Z\sim\mathcal{N}(0,1) \\ \Rightarrow \mathbb{P}(Y \gt 0)= 1-\Phi\left(\frac{-\mu}{\sigma}\right) \\ \Rightarrow X_i \sim Ber\left(1-\Phi\left(\frac{-\mu}{\sigma}\right)\right) \]
We have lots of data (or say \(X_i\)'s) so if we want to identify the parameter of \(X_i\) which is a function of \(\mu\) and \(\sigma\), we took average of \(X_i\)'s that is \(\overline{X}_n\)
So \(\overline{X}_n = 1-\Phi\left(\frac{-\mu}{\sigma}\right)\)
\(\Rightarrow \frac{\mu}{\sigma}=-\Phi^{-1}(1-\overline{X}_n)\)
So we can estimate \(\frac{\mu}{\sigma}\) but we cannot estimate \((\mu,\sigma^2)\) as a pair.
Say that we estimated \(\frac{\mu}{\sigma} = \frac{1}{2}\)
So we can say that \( (\mu,\sigma^2) =(1,4)\) but we can also say that \( (\mu,\sigma^2) =(2,16)\)
Now we can't decide between them.
Mathematically: say \(\theta=(\mu,\sigma^2)=(1,4)\) and \(\theta'=(\mu',(\sigma')^2)=(2,16)\)
and \(\theta \neq \theta'\) but \(\mathbb{P}_\theta = \mathbb{P}_{\theta'}\)
  • \( \mathbb{P}_\theta = Ber\left(1-\Phi\left(-\frac{1}{2}\right)\right) \)
    \( \mathbb{P}_{\theta'} = Ber\left(1-\Phi\left(-\frac{2}{4}\right)\right) \)
    \(\Rightarrow \mathbb{P}_\theta=\mathbb{P}_{\theta'}\)
So our map \( \theta \in \Theta \mapsto \mathbb{P}_\theta \) is not injective
So \( (\mu,\sigma^2)\) as a pair is not identifiable
What is our sample space \(E\) \(E=\{0,1\}\)
What is the \((\mathbb{P}_\theta)_{\theta\in\Theta}\) Here we have two choices:
  1. \[ (\mathbb{P}_\theta)_{\theta\in\Theta} =Ber\left(1-\Phi\left(\frac{-\mu}{\sigma}\right)\right)_{(\mu,\sigma^2)\in\mathbb{R}\times (0,\infty)} \] In this case the parameter \(\theta=(\mu,\sigma^2)\) is not identifiable.
  2. \[ (\mathbb{P}_\theta)_{\theta\in\Theta} =Ber\left(1-\Phi\left(\frac{-\mu}{\sigma}\right)\right)_{\frac{\mu}{\sigma}\in \mathbb{R}} \] In this case the parameter \(\theta=\frac{\mu}{\sigma}\) is identifiable.
Which of them is a valid statistical model?
\( \left( \{0\}, (Ber(p))_{p \in (0,1)} \right) \)
It is not a valid statistical model because sample space of a Bernoulli r.v. is \(E=\{0,1\}\)
\( \left( \{0,1\}, (Ber(p))_{p \in (0.2,0.3)} \right) \)
It is a valid statistical model sample space is correct \(E=\{0,1\}\) and parameter space is also correct \(p \in (0.2,0.3)\) because \((0.2,0.3)\in(0,1)\) so it's correct

Let \(X_1, \cdots ,X_n \stackrel{iid}{\sim } \mathcal{U}([0,a])\) is a Uniform r.v. on interval \([0,a]\) for some unknown \(a \gt 0\).
According to given situation which of the following is a valid statistical model?
\( \left( [0,a], (\mathcal{U}([0,a]))_{a \gt 0} \right) \)
It is not a valid statistical model because sample space is not defined, sample space \(E=[0,a]\) depends on \(a\) which is not known.
So our model is not valid.
\( \left( R_+, (\mathcal{U}([0,a]))_{a \gt 0} \right) \)
It is a valid statistical model because sample space is well defined, sample space \(E=(0,\infty]\) it does not depends on \(a\).
So our model is valid.
Rule of thumb: Our sample space should not depend on the parameter
