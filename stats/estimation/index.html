When we encounter a Statistical inference problem first thing we encounter is Estimation, we gather our data \((X_1,\cdots,X_n)\), if our model is well specified then it comes from a particular family of distribution, and if our model is also identifiable then we can estimate it.
Definitions:

  • Statistic is any measurable function \(f(X_1,\cdots,X_n)\) of the sample
    for example: \(\overline{X}_n,\ \text{max}_i X_i,\ X_n,\ \overline{X}_ne^{X_1},\) median, variance \(,\cdots\)
    • Rule of thumb: if you can compute if (once given data) it's measurable
  • Estimator of \(\theta\) is any statistics whose expression does not depends on the parameter \(\theta\)
  • Once we have an estimator \(\hat{\theta}_n\), we go further for it's properties.
    Properties:
    1. As we collect more and more data, we want our estimator \(\hat{\theta}_n\) to be close to the true parameter \(\theta\).
      Estimator \(\hat{\theta}_n\) of \(\theta\) is consistent if:
      \[\hat{\theta}_n \xrightarrow [n\to \infty ]{\mathbb{P,\text{ a.s.}}} \theta\]
    2. Estimator \(\hat{\theta}_n\) of \(\theta\) is asymptotically normal if: \[\sqrt{n} (\hat{\theta}_n-\theta ) \xrightarrow [n\to \infty ]{(d)} \mathcal{N}(0,\sigma^2)\] And the quantity \(\sigma^2\) is called asymptotic variance of \(\hat{\theta}_n\)
