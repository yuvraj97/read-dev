In Hypothesis testing we are asking binary questions like:

  • In our kiss example we were try to answer a binary question that do people have a preference to turn there head in a particular direction
  • In our previous coin toss example we are questioning about the fairness of coin.
    • So we expect a binary answer, now we will define a function that takes our data and output \(0\) if we want to conclude \(H_0\) null hypothesis or \(1\) if we want to conclude \(H_1\) alternative hypothesis.

    Statistical test

    A test is a statistics \(\psi\in\{0,1\}\) such that
  • If \(\psi=0,\) we failed to reject \(H_0\) (null hypothesis)
  • If \(\psi=1,\) we reject \(H_0\) (null hypothesis)
    • In our coin toss example we are questioning about the fairness of coin. There is no reason for it to be unfair, so
    \( \begin{matrix} \Theta_0=\{1/2\} &;\ H_0:\quad p=1/2 \\ \Theta_1=(0,1)-\{1/2\} &;\ H_1:\quad p\neq 1/2 \\ \end{matrix} \)

    Now what should be our test , \(\psi=?\)
    \(\psi\) outputs either \(0\) or \(1\), so \(\psi\) should be an indicator function.
    but indicator of what \(\psi=𝟙(?)\).
    When we want \(\psi\) to be equals to \(1\)?
    \(\psi=1\) means we reject null hypothesis(status quo) \(\Rightarrow p\neq 1/2 \Rightarrow\) that our data shows that the coin is biased.
    So \(\psi=1\) when \(\overline{X}_n\) is significantly far from \(1/2\)
    \(|\overline{X}_n-1/2|\gt C\), for some \(C>0\)
    \[\psi=𝟙\left\{\sqrt{n}\frac{|\overline{X}_n-1/2|}{1/2(1-1/2)} \gt C\right\};\quad \text{for some } C>0\]
    Now question arises that how to choose this threshold \(C\)?
    To answer it first we need to understand what we are optimizing.
    When a test output \(0\) or \(1\) it can make error, and we can measure performance of a test\((\psi)\) by it's error
    Error can be like:
  • Test\((\psi)\) outputs \(0\Rightarrow\) it failed to reject \(H_0\) (null hypothesis) but \(H_0\) is indeed false it's an error.
  • Test\((\psi)\) outputs \(1\Rightarrow\) it reject \(H_0\) (null hypothesis) but \(H_0\) is indeed true it's an error.
    • Some test can make this error again and again, mean if you provide some different sample, test will still make an error.
    For Example
    Say \(C=0\) now the test\((\psi)\) will constantly make error, because \(\overline{X}_n\) will deviate from \(1/2\) even if there is a slightest deviation (well it's not impossible for \(|\overline{X}_n-1/2|\) to be equal to \(0\) but chances of it is very slim.) so then \(|\overline{X}_n-1/2|\) will always be \(\gt 0\) so this test will always reject.
    Now we have to find the \(C\) that minimizes these errors.

    Errors

    Rejection Region

    A sample of \(n\) observations is something like \(X_1,X_2,X_3,....,X_n\) and they are \(i.i.d\) random variables.
    Think of a space where \(X_1,X_2,X_3,....,X_n\) lives it's a \(n\) dimensional space, we denote this \(n\) dimensional space as \(E^n\), so every point in this space is a collection of \(n\) i.i.d. random variables.
    A Rejection region(\(R_\psi\)) of a test \(\psi\) is the the subspace of this \(n\) dimensional space, such that, \(\psi(x) = 1;\quad x\in E^n\)
    \[R_\psi = \{ x\in E^n : \psi(x)=1\}\] So Rejection region is a subset of all possible samples that would lead to a rejection.

    Types of error

    There are \(2\) types of error that can occur:
  • Type 1 Error
    If \(H_0\) is true but our test\((\psi)\) rejects \(H_0\), that mean \(\psi = 1\)
  • Type 2 Error
    If \(H_0\) is false but our test\((\psi)\) fail to rejects \(H_0\), that mean \(\psi = 0\)

    • \( \begin{matrix} &\\ & \text{Test }\psi \\ \text{Reality}& \\ \end{matrix} \) \(H_0\ (\psi = 0)\) \(H_1\ (\psi = 1)\)
    \(H_0\ (\theta\in\Theta_0)\) \(\color{green}{\text{No Error}}\) \(\color{red}{\text{Type 1 Error}}\)
    \(H_1\ (\theta\in\Theta_1)\) \(\color{red}{\text{Type 2 Error}}\) \(\color{green}{\text{No Error}}\)
    Type \(1\) error of a test \(\psi\)
    Type 1 error occur if \(H_0\) is true but our test(\(\psi\)) rejects \(H_0\), that mean \(\psi = 1\)
    It is denoted br \(\alpha_\psi\).
    \(H_0:\quad \theta\in\Theta_0\)
    We quantify Type 1 error by it's probability of occurrence provided the true parameter \(\theta\).
    This probability will be a function of \(\theta\).
    \[ \begin{matrix} \alpha_\psi &:& \Theta_0 & \to & [0,1] \\ & & \theta & \mapsto & \mathbb{P}_{\theta} [\psi = 1]\\ \end{matrix} \]
    Type \(2\) error of a test \(\psi\)
    Type 2 error occur if \(H_0\) is false but our test(\(\psi\)) fail to rejects \(H_0\), that mean \(\psi = 0\)
    It is denoted br \(\beta_\psi\).
    \(H_1:\quad \theta\in\Theta_1\)
    We quantify Type 2 error by it's probability of occurrence provided the true parameter \(\theta\).
    This probability will also be a function of \(\theta\).
    \[ \begin{matrix} \beta_\psi &:& \Theta_1 & \to & [0,1] \\ & & \theta & \mapsto & \mathbb{P}_{\theta} [\psi = 0]\\ \end{matrix} \]
    Power of a Test \(\psi\)
    It's the probability when we correctly reject, that is \(H_0\) is false and our test\((\psi)\) rejects the null hypothesis \(\psi =1\)
    And Power is minimum probability of correctly rejecting the null, among all possible \(\theta\in\Theta_1\).
    \[\pi_\psi = \mathop {\rm inf}\limits _{\theta\in\Theta_1}(1-\beta_\psi(\theta))\]
    Say \(\Theta_0=[0,1]\) and \(\Theta_1=[1,\infty]\)
    Now let's see how \(\mathbb{P}_{\theta} [\psi = 1]\) and \(\theta\) interact.
    When \(\theta\in\Theta_0\) we want to reject as minimum as possible, we want \(\mathbb{P}_{\theta} [\psi = 1]\) to be minimum.
    When \(\theta\in\Theta_1\) we want to reject as much as possible, we want \(\mathbb{P}_{\theta} [\psi = 1]\) to be maximum.


    In this image blue line is the asymptote and the red line is \((1-\beta_\psi)\).
    And the threshold \(C\) will decide \(\pi_\psi\), \(C\) will decide our trade-off between Type 1 and Type 2 error.

    Level

    Level is an upper bound on the Type 1 Error.
    Say we want a test with level \(5\%\) what we are saying is we don't allow our Type 1 Error to exceed \(5\%\).
  • A test \(\psi\) has level \(\alpha\) if
    • \[\alpha_\psi(\theta) \leq \alpha;\quad \forall \theta\in\Theta_0\] Remember our     Criminal Trial example,
    where the null hypothesis is that the person is not guilty.
    And alternative hypothesis is that the person is guilty.
    Here Type 1 Error says "If the person is not guilty" we want the probability of "that person to go to jail" to be at most \(5\%\).
    How can we have \(0\) Type \(1\) Error?
    To achieve \(0\) Type \(1\) Error, our test is always \(0\), \(\psi = 0\) which mean, null hypothesis is always true no matter what.
    So let everyone go no matter that the person is guilty or not.
    How can we have \(0\) Type \(2\) Error?
    To achieve \(0\) Type \(2\) Error, our test is always \(1\), \(\psi = 1\) which mean, alternative hypothesis is always true no matter what.
    So send everyone to jail.
    We cannot make any of the error \(=0\), we have to set a threshold, we have to a make a trade off.
    So we will control one of the error(either Type 1 or Type 2) and we cross our fingers for other one.
    Here we are focusing to control Type 1 error.
    Like for the Criminal Trial example if we says we want Type 1 error at most \(5\%\), it meant that at most \(5\%\) of the non guilty person goes to Jail, because we aren't controlling Type 2 error we have to let, some guilty person go free.

    We can argue that we want Type 1 error at most \(5\%\) and \(0\% \lt 5\%\) so let everyone go free.
    That's why we want to make this inequality an equality.

  • A test \(\psi\) has asymptotic level \(\alpha\) if
    • \[ \mathop {\rm lim}\limits _{n \to \infty} \alpha_{\psi_n}(\theta) \leq\alpha;\quad \forall \theta\in\Theta_0\] Here our test note depends on our sample size \(n\).
  • In general test is of form
    • \[\psi = 𝟙\{T_n\gt C\};\quad C\in\mathbb{R}\] for some Statistics \(T_n\)

    Type of Tests

    In Hypothesis Testing we can have two kind of tests, One sided test or Two sided test.

  • One sided test
    • Here we test "is our parameter \(\theta\) is less then or greater then some constant \(\theta_0\)"
    If \(H_1: \theta \lt \theta_0\) or \(H_1: \theta \gt \theta_0\) the test is one sided.
  • Two sided test
    • Here we test "is our parameter \(\theta\) is equals to some constant \(\theta_0\) or not"
    If \(H_1: \theta \neq \theta_0\) the test is two sided.
    Examples
  • Kiss example:
    • Here we are testing "do people have any preference to turn their head to left or right"
    \(H_0: p=1/2; \quad H_1: p \neq 1/2\)
    It's Two sided test
  • R2F vs WilMA
    • Here we are testing "is WilMA better then R2F, or there is no difference."
    \(H_0: \mu_{\text{WilMA}} = \mu_{\text{R2F}},\quad \) \(H_1: \mu_{\text{WilMA}} \lt \mu_{\text{R2F}} \)
    It is One sided test.
    If we change our statement and ask "is there any difference between WilMA and R2F is one is better then the other." then,
    \(H_0: \mu_{\text{WilMA}} = \mu_{\text{R2F}},\quad \) \(H_1: \mu_{\text{WilMA}} \neq \mu_{\text{R2F}} \)
    Now it becomes Two sided test
  • Coin Toss Example
    • Here we are testing "is our coin is fair, or it's biased toward one of the side"
    \(H_0: p=1/2; \quad H_1: p \neq 1/2\)
    It's Two sided test
  • YouTube Example
    • Here we are testing "is there at most \(1/3\) of all Americans get at least some of there news on YouTube, or is it less then \(1/3\)
    \(H_0: p\geq 0.33 ;\quad H_1: p\lt 0.33\)"
    It's One sided test

    Examples

    Coin toss example

    Recall our Coin toss example
    Here we flipped a coin \(50\) times and it land heads \(23\) times, and we are asking about the fairness of the coin.
    Here our null hypothesis \(H_0\) is: The coin is fair, \(p=1/2\)
    Here our alternate hypothesis \(H_1\) is: The coin is not fair, \(p\neq 1/2\)
    Can we reject the null hypothesis at asymptotic level \(\alpha=5\%\)?

    Modelling Assumptions
    We have \(n\) observations.
    So we have \(n\) i.i.d. r.v. with Bernoulli distribution.
    \(X_1,...,X_n\stackrel{iid}{\sim}\text{Ber}(p);\quad p\in(0,1)\)
    \[X_i=\left\{\begin{matrix} \ \ 1\quad\text{;if ith outcome is Heads} \\ 0\quad\text{;if ith outcome is Tails} \end{matrix}\right.\]
    Estimator
    Estimator of \(p\):
    \[\hat{p}_n = \frac{1}{n}\sum_{i=1}^nX_i =\overline{X}_n\]
    Test statistics
    \(H_0: p=1/2; \quad H_1: p \neq 1/2\)
    So we want to test the deviation of our estimate \(\hat{p}_n\) from true parameter \(p\)
    We can get the deviation as, \(|\hat{p}_n-p|\), and then we can say if \(|\hat{p}_n-p|\) is far away from \(0\) then according to our data true parameter is unlikely to be \(p\), but how much far away?

    \(\mathbb{E}[\hat{p}_n-p] = 0,\quad\) \(\text{var}(\hat{p}_n-p) = \text{var}(\hat{p}_n) = \frac{p(1-p)}{n}\)
    Then by CLT \[\sqrt{n}\frac{\hat{p}_n-p}{\sqrt{p(1-p)}} \xrightarrow [n\to \infty ]{(d)} \mathcal{N}(0,1)\] We will reject the null hypothesis \(H_0\) if,
    \(|\hat{p}_n-p|\gt C;\quad C\in\mathbb{R}^+\)
    \(\Rightarrow \sqrt{n}\frac{|\hat{p}_n-p|}{\sqrt{p(1-p)}} \gt \sqrt{n}\frac{C}{\sqrt{p(1-p)}} \)

    \(\Rightarrow \sqrt{n}\frac{|\hat{p}_n-p|}{\sqrt{p(1-p)}} \gt C';\quad C\in\mathbb{R}^+\)
    So we define our test statistics to be,
    \[T_n = \sqrt{n}\frac{|\hat{p}_n-p|}{\sqrt{p(1-p)}} \]
    Test
    \[\psi =𝟙\{T_n\gt C\};\quad C\in\mathbb{R}\] Here we want our type 1 error to be equal to \(\alpha\).
    We want \(\mathbb{P}_{p} [\psi = 1] \xrightarrow [n\to \infty ]{} \alpha\) over all \(p\in\Theta_0\).
    But in our example \(\Theta_0=\{1/2\}\).
    So we want \(\mathbb{P}_{1/2} [\psi = 1] \xrightarrow [n\to \infty ]{} \alpha\) and here \(\alpha\) is our asymptotic level
    \[\Rightarrow \mathbb{P}_{1/2} \left[ \sqrt{n}\frac{|\hat{p}_n-1/2|}{\sqrt{1/2(1-1/2)}} \gt C \right] \xrightarrow [n\to \infty ]{} \alpha\] \[\Rightarrow \mathbb{P}_{1/2} \left[ \sqrt{n}\frac{|\hat{p}_n-1/2|}{\sqrt{1/2(1-1/2)}} \gt q_{\alpha/2} \right] \xrightarrow [n\to \infty ]{} \alpha\] And \(q_{\alpha/2}\) is quantile of order \(1- \alpha/2\) of a r.v. \(\mathcal{N}(0,1)\), we discussed it in confidence interval
    So, \[\psi =𝟙\{T_n\gt q_{\alpha/2}\}\]
    Estimator of \(p\): \(\quad \hat{p}_n = \frac{1}{n}\sum_{i=1}^nX_i =\overline{X}_n = 23/50\)
    \(T_n=\sqrt{n}\frac{|\overline{X}_n-0.5|}{0.5(1-0.5)} =0.56\)
    \(\psi =𝟙\{T_n\gt q_{\alpha/2}\}\)
    \(q_{\alpha/2}=q_{0.025}=1.96\) (we discussed it here)
    And \(0.56 \ngtr 1.96\) so \(\psi = 0\)
    So we failed to reject the null hypothesis.

    YouTube example

    Recall our YouTube example
    Here the claim is among \(4,971\) randomly samples Americans , \(32\%\) report to get at least some of there news on YouTube.
    Here our null hypothesis \(H_0\) is: at most \(1/3\) of all Americans get at least some of there news on YouTube
    Here our alternate hypothesis \(H_1\) is: at most less then \(1/3\) of all Americans get at least some of there news on YouTube
    \(H_0: p\geq 0.33 ;\quad H_1: p\lt 0.33\)
    Can we reject the null hypothesis at asymptotic level \(\alpha=5\%\)?

    Modelling Assumptions
    We have \(n=4,971\) observations.
    So we have \(n\) i.i.d. r.v. with Bernoulli distribution.
    \(X_1,...,X_n\stackrel{iid}{\sim}\text{Ber}(p);\quad p\in(0,1)\)
    \[X_i=\left\{\begin{matrix} 1\quad\text{;if ith person get some of it's news from YouTube}\quad\quad \\ 0\quad\text{;if ith person do not get any of it's news from YouTube} \end{matrix}\right.\]

    Estimator
    Estimator of \(p\):
    \[\hat{p}_n = \frac{1}{n}\sum_{i=1}^nX_i =\overline{X}_n\]
    Test statistics
    \(H_0: p\geq 0.33 ;\quad H_1: p\lt 0.33\)
    If our estimator \(\hat{p}_n \) is less enough then true parameter \(p\) then we reject the null hypothesis.
    In order to reject the null hypothesis we want,
    \(\hat{p}_n -p \lt C; \quad C\in\mathbb{R}^+\)
    \(\mathbb{E}[\hat{p}_n-p] = 0,\quad\) \(\text{var}(\hat{p}_n-p) = \text{var}(\hat{p}_n) = \frac{p(1-p)}{n}\)
    Then by CLT \[\sqrt{n}\frac{\hat{p}_n-p}{\sqrt{p(1-p)}} \xrightarrow [n\to \infty ]{(d)} \mathcal{N}(0,1)\]
    \(\Rightarrow \sqrt{n}\frac{\hat{p}_n -p}{\sqrt{p(1-p)}} \lt \sqrt{n}\frac{C}{\sqrt{p(1-p)}}\)

    \(\Rightarrow \sqrt{n}\frac{\hat{p}_n -p}{\sqrt{p(1-p)}} \lt C'; \quad C'\in\mathbb{R}^+\)
    So our Test statistics will be,
    \[T_n = \sqrt{n}\frac{\hat{p}_n-p}{\sqrt{p(1-p)}} \]
    Test
    \[\psi =𝟙\{T_n\lt C\};\quad C\in\mathbb{R}^+\] Here we want our Type 1 error \( \xrightarrow [n\to \infty ]{} \alpha\).
    \(\mathbb{P}_{p} [\psi = 1] \xrightarrow [n\to \infty ]{} \alpha(p)\) over all \(p\in\Theta_0\).
    And we want \(\alpha(p)\) to not to exceed \(\alpha; \quad\) \(\forall p\in\Theta_0\)
    Here \(\Theta_0 =[0.33,1)\)
    \[\mathbb{P}_{p}\left[ \sqrt{n}\frac{\hat{p}_n-p}{\sqrt{p(1-p)}} \lt C \right] \xrightarrow [n\to \infty ]{} \alpha(p)\]
    \[\mathop {\rm max}\limits _{p\in\Theta_0} \alpha(p) \xrightarrow [n\to \infty ]{} \alpha\]
    Notice that in previous example we have a singleton \(\Theta_0\) so we just test for that particular value of \(\Theta_0\)
    But here we have a whole range for the value of \(p\), so our test must reject for all of those \(p\)'s.
    \(C\) is a constant, so as \(p\) changes \(\alpha(p)\) also changes, different \(p\) results in different Type 1 Error.
    Say \(p'=\mathop {\rm arg~ max}\limits _{p\in\Theta_0}\alpha(p)\quad\) so
    \(\mathbb{P}_{p'}\left[ \sqrt{n}\frac{\hat{p}_n-p'}{\sqrt{p'(1-p')}} \lt C \right] \xrightarrow [n\to \infty ]{} \alpha\)
    and \(\alpha(p)\leq\alpha;\quad \forall p\in\Theta_0\) and \(\alpha\) is the level of our test.
    So if our test \(\psi\) is rejected for \(p'\) mean \(\psi=1\), then test \((\psi)\) will be rejected \(\forall\ p\in\Theta_0\)
    \[\mathbb{P}_{p}\left[ \sqrt{n}\frac{\hat{p}_n-p}{\sqrt{p(1-p)}} \lt C \right] \xrightarrow [n\to \infty ]{} \alpha\]
    \(\Rightarrow \mathbb{P}( \mathcal{N}(0,1) \lt C) \xrightarrow [n\to \infty ]{} \alpha\)
    It's means CDF of \(\mathcal{N}(0,1)\) at \(C\) goes to \(\alpha\) as \(n\to \infty\)
    \(\Rightarrow \Phi(C) \xrightarrow [n\to \infty ]{} \alpha\)
    \(\Rightarrow C = \Phi^{-1}(\alpha)\)
    And it is quantile of order \(1-\alpha\) of a r.v. \(\mathcal{N}(0,1)\) denoted as   \(q_{1-\alpha}\)
    (we discussed it in confidence interval)
    So our Test becomes: \[\psi =𝟙\{T_n\lt q_{1-\alpha}\}\] And \(q_{1-\alpha}=-q_{\alpha}\) \[\psi =𝟙\{T_n\lt -q_{\alpha}\}\]
    \[\mathbb{P}_{p}\left[ \sqrt{n}\frac{\hat{p}_n-p}{\sqrt{p(1-p)}} \lt q_{1-\alpha} \right] \xrightarrow [n\to \infty ]{} \alpha\]
    Now we want to find \(p'\) such that \(p'=\mathop {\rm arg~ max}\limits _{p\in\Theta_0}\alpha(p)\) or say "\(p'\) is \(p\) that gives us maximum Type 1 Error"
    \(\Theta_0 = [0.33,1)\) so \(p'=0.33\)
    We want our Type 1 Error do not exceed \(\alpha\).
    Here we assume that null hypothesis is true and we test for alternative hypothesis.
    we are testing: \(\hat{p}_n-p \lt C;\quad p\in\Theta_0\) and \(\Theta_0 = [0.33,1)\)
    So if \(\hat{p}_n-0.33 \lt C\) then \(\hat{p}_n-p \lt C;\quad\forall p\in\Theta_0\) because \(p\geq 0.33\)

    But why \(0.33=\mathop {\rm arg~ max}\limits _{p\in\Theta_0}\alpha(p)\)
    Here \(p\) is our true parameter \((p\in\Theta_0)\) (which we don't know) and \(\hat{p}_n\) is the estimate of \(p\) based on a single sample, if we perform the experiment multiple times, then sometime \(\hat{p}_n\) falls in left of \(p\) and sometime if falls on right, means sometime \(\hat{p}_n\lt p\) and sometime \(\hat{p}_n\gt p\)
    If we choose \(p=1\) then it's estimator \(\hat{p}_n\) will rarely be less then \(0.33\) means it will rarely reject the null(make an error)
    If we choose \(p=0.75\) then it's estimator \(\hat{p}_n\) will less rarely be less then \(0.33\) means we have some better chances to reject the null (make an error)
    If we choose \(p=0.5\) then it's estimator \(\hat{p}_n\) will more likely be less then \(0.33\) means we have much better chances to reject the null (make an error)
    If we choose \(p=0.33\) then it's estimator \(\hat{p}_n\) will highly likely be less then \(0.33\) means we have very high chances to reject the null (make an error)
    So \(p=0.33=\mathop {\rm arg~ max}\limits _{p\in\Theta_0}\alpha(p)\)
    According to our YouTube example \(T_n=-1.5\)
    \[\psi =𝟙\{T_n\lt -q_{\alpha}\}\] \(\alpha=0.05\Rightarrow\) \(q_{0.05}=1.645\)
    And \(-1.5 \nless -1.645\) So we failed to reject the null Hypothesis.
