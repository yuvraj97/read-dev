Bias of an estimator

Now we know the properties of an estimator \(\hat{\theta}_n\) of \(\theta\)
\(\hat{\theta}_n\) is a r.v. and we hope it to be centered about \(\theta\), so we look at it's expectation.
\(\text{bias}( \hat{\theta}_n ) = \mathbb{E}[ \hat{\theta}_n ] - \theta \)
If \(\text{bias}=0 \Rightarrow \hat{\theta}_n\) is unbiased
We have to be very careful when choosing an estimator, unbiased estimators are not always the best estimators that we can have, maybe the unbiased estimator comes with a large variance and at same time a very little biased estimator comes with a smaller variance.
So unbiased estimators are not always the best estimator.
Examples: Here we assume that \(X_1,\cdots,X_n \stackrel{iid}{\sim } Ber(p)\) now we compute the \(\text{bias}\) for the following estimators:
\( \hat{p}_n = \overline{X}_n\)
\(\text{bias}(\hat{p}_n) = \mathbb{E}[\hat{p}_n] -p\)
\(\mathbb{E}[\hat{p}_n]=\frac{1}{n}(\mathbb{E}[X_1]+\cdots+\mathbb{E}[X_n])=p\)
\(\Rightarrow \text{bias}(\hat{p}_n)=p-p=0\)
So the estimator is unbiased.
\( \hat{p}_n = X_1\)
\(\text{bias}(\hat{p}_n) = \mathbb{E}[\hat{p}_n] -p\)
\(\text{bias}(\hat{p}_n) = \mathbb{E}[X_1] -p\)
\(\text{bias}(\hat{p}_n) = p-p=0\)
So the estimator is unbiased, but this estimator does not make any sense \(X\) is either \(0\) or \(1\) so our estimator is also either \(0\) or \(1\).
So this estimator is unbiased but useless.
\( \hat{p}_n = \frac{X_1+X_2}{2}\)
\(\text{bias}(\hat{p}_n) = \mathbb{E}[\hat{p}_n] -p\)
\(\text{bias}(\hat{p}_n) = \frac{\mathbb{E}[X_1]+\mathbb{E}[X_2]}{2} -p\)
\(\text{bias}(\hat{p}_n) = p-p=0\)
So the estimator is unbiased, but this estimator also does not make any sense, we have a lots of data and we are using only two outcomes. So this estimator is unbiased but useless.
\( \hat{p}_n = \sqrt{𝟙(X_1=1 , X_2=1)}\)
\( \hat{p}_n \sim \sqrt{Ber(\mathbb{P}(X_1=1)\times \mathbb{P}(X_2=1)) } \)
\( \hat{p}_n \sim \sqrt{Ber(p^2)} \)
  • Now if we proceed to evaluate \(\text{bias}(\hat{p}_n)\)
    \(\text{bias}(\hat{p}_n) = \mathbb{E}\left[\sqrt{Ber(p^2)}\right] -p \)
    and \(\mathbb{E}\left[\sqrt{Ber(p^2)}\right]\) is something we don't knew.
    but for Bernoulli \( \sqrt{Ber(p^2)} \sim Ber(p^2)\)
\( \hat{p}_n \sim Ber(p^2) \)
\(\text{bias}(\hat{p}_n) = \mathbb{E}[\hat{p}_n] -p = p^2 - p\)

Jensen's Inequality

According to Jensen's Inequality, a function \(f\) which is:
  1. Convex:

    For a convex function the chord is always above the function so:
    \[E[f(X)] \geq f(E[X])\]
  2. Concave:

    For a concave function the chord is always below the function so:
    \[E[f(X)] \leq f(E[X])\]

Example: \(f(x) = \sqrt{x}\) is concave, so:
\(E\left[\sqrt{Z}\right] \leq \sqrt{E[Z]}\), \(Z\) can be any distribution.
Say \(Z\sim Ber(p^2)\) as in our example given above, so:
\(E\left[\sqrt{Ber(p^2)}\right] \leq \sqrt{E[Ber(p^2)]}\)
  • And as we seen in above example
    \(\sqrt{Ber(p^2)} \sim Ber(p^2)\)
\(\Rightarrow E[Ber(p^2)] \leq \sqrt{E[Ber(p^2)]}\)
\(\Rightarrow p^2 \leq \sqrt{p^2}\)
\(\Rightarrow p^2 \leq p\) which is indeed true

Variance of an estimator

Estimator is a random variable so it must have it's variance.
\(Var(X)=\mathbb{E}\left[ (X-\mathbb{E}[X])^2 \right] = \mathbb{E}[X^2] - (\mathbb{E}[X])^2 \)
Examples:
\( \hat{p}_n = \overline{X}_n\)
\(Var(\hat{p}_n) = \frac{1}{n}(Var[X_1]+\cdots+Var[X_n])\)

\(X_i \sim Ber(p)\) so \(Var(X_i) =p(1-p)\)
\(\Rightarrow Var(\hat{p}_n) =\frac{p(1-p)}{n}\)
So \(Var(\hat{p}_n) \xrightarrow [n\to \infty ]{} 0 \)
What is the maximum variance if \(n\) is kept constant>
We can get max or min of a function \(f\) by equating its derivative \(f'\) equals to \(0\)
\(f(p) = p(1-p)\)
\(f'(p)=(1-p)+p(-1)\)
equating \(f'(p) = 0\)
\(\Rightarrow 1-2p=0 \Rightarrow p=1/2\)
So that maximum is achieved at \(p=1/2\) and maximum variance is \(Var(1/2)=\frac{1}{n}1/2(1-1/2)=\frac{1}{4n}\)
\( \hat{p}_n = X_1\)
\(Var(\hat{p}_n) = Var(X_1) = p(1-p)\)
As we discussed above that this estimator does not make any sense
\(X\) is either \(0\) or \(1\) so our estimator is also either \(0\) or \(1\).
So this estimator is useless.
\( \hat{p}_n = \frac{X_1+X_2}{2}\)
\(Var(\hat{p}_n) = Var\left(\frac{X_1 + X_2}{2}\right) \)
\(Var(\hat{p}_n) = \frac{Var(X_1) + Var(X_2)}{4}\)
\(Var(\hat{p}_n) = \frac{p(1-p)}{2}\)
As we discussed above that this estimator also does not make any sense, we have a lots of data and we are using only two outcomes. So this estimator is useless.
\( \hat{p}_n = \sqrt{𝟙(X_1=1 , X_2=1)}\)
\( \hat{p}_n \sim \sqrt{Ber(\mathbb{P}(X_1=1)\times \mathbb{P}(X_2=1)) } \)
\( \hat{p}_n \sim \sqrt{Ber(p^2)} \)
  • Now if we proceed to evaluate \(\text{bias}(\hat{p}_n)\)
    \(\text{bias}(\hat{p}_n) = \mathbb{E}\left[\sqrt{Ber(p^2)}\right] -p \)
    and \(\mathbb{E}\left[\sqrt{Ber(p^2)}\right]\) is something we don't knew.
    but for Bernoulli \( \sqrt{Ber(p^2)} \sim Ber(p^2)\)
\( \hat{p}_n \sim Ber(p^2) \)
So \(Var(\hat{p}_n) = p^2(1-p^2) \)


Quadratic Risk of an estimator

While selecting an estimator we want our estimator to have minimum \(\text{bias}\) and minimum \(Var\)iance
We can select among estimators which has minimum \(\text{bias}\) or minimum \(var\)iance individually.
But we want \(\text{bias}\) and \(var\)iance to be minimum simultaneously, so how to do it?
And the answer is Quadratic Risk.
Quadratic Risk
Here we are trying to minimize \(\text{bias}\) and \(var\)iance at the same time.
And rather then saying that minimize the product of \(\text{bias}\) and \(var\)iance or minimize the sum of \(\text{bias}\) and \(var\)iance.
What we say is \(\hat{\theta}_n \xrightarrow [n\to \infty ]{} \theta \), or say \(\hat{\theta}_n -\theta \xrightarrow [n\to \infty ]{} 0 \)
So we want our random variable \((\hat{\theta}_n -\theta )\) to be small as \(n\to \infty\)
Quadratic Risk \(R\left( \hat{\theta}_n \right)\) of an estimator \(\hat{\theta}_n \in \mathbb{R}\) is:
\[R \left( \hat{\theta}_n \right) = \mathbb{E} \left[ \left( \hat{\theta}_n - \theta \right)^2 \right] \] Now lets see how minimizing the Quadratic Risk minimizes our \(\text{bias}\) and \(var\)iance:
\(R \left( \hat{\theta}_n \right) = \mathbb{E} \left[ \left( \hat{\theta}_n - \theta \right)^2 \right]\)
\(R \left( \hat{\theta}_n \right) = \mathbb{E} \left[ \left( \hat{\theta}_n - \mathbb{E}\left[\hat{\theta}_n \right] +\mathbb{E}\left[\hat{\theta}_n \right] - \theta \right)^2 \right]\)
\(R \left( \hat{\theta}_n \right) = \mathbb{E} \left[ \left( \hat{\theta}_n - \mathbb{E}\left[\hat{\theta}_n \right] \right)^2 + \left( \mathbb{E}\left[\hat{\theta}_n \right] - \theta \right)^2 +2\left( \hat{\theta}_n - \mathbb{E}\left[\hat{\theta}_n \right] \right) \left( \mathbb{E}\left[\hat{\theta}_n \right] - \theta \right) \right] \)
\(R \left( \hat{\theta}_n \right) = \underbrace{\mathbb{E} \left[ \left( \hat{\theta}_n - \mathbb{E}\left[\hat{\theta}_n \right] \right)^2 \right]}_{Var(\hat{\theta}_n)} + \mathbb{E}\left[ \left( \mathbb{E}\left[\hat{\theta}_n \right] - \theta \right)^2 \right] \)
\(\quad\quad\quad\quad + 2 \mathbb{E} \left[ \left( \hat{\theta}_n - \mathbb{E}\left[\hat{\theta}_n \right] \right) \left( \mathbb{E}\left[\hat{\theta}_n \right] - \theta \right) \right]\)

\(\mathbb{E}\left[\hat{\theta}_n \right]\) and \(\theta\) are not a random variable they are deterministic so \( \left( \mathbb{E}\left[\hat{\theta}_n \right] - \theta \right)\) is not a r.v. so \(\mathbb{E}\left[ \left( \mathbb{E}\left[\hat{\theta}_n \right] - \theta \right)\right] =\)   \(\left( \mathbb{E}\left[\hat{\theta}_n \right] - \theta \right)\)
And \(\left( \mathbb{E}\left[\hat{\theta}_n \right] - \theta \right) = \text{bias}\left( \hat{\theta}_n \right) \)

\(R \left( \hat{\theta}_n \right) = Var(\hat{\theta}_n) + \text{bias}^2(\hat{\theta}_n) + 2 \mathbb{E} \left[ \left( \hat{\theta}_n - \mathbb{E}\left[\hat{\theta}_n \right] \right) \left( \mathbb{E}\left[\hat{\theta}_n \right] - \theta \right) \right] \)
\(R \left( \hat{\theta}_n \right) = Var(\hat{\theta}_n) + \text{bias}^2(\hat{\theta}_n) + 2 \left( \mathbb{E}\left[\hat{\theta}_n \right] - \theta \right) \underbrace{\mathbb{E} \left[ \left( \hat{\theta}_n - \mathbb{E}\left[\hat{\theta}_n \right] \right) \right]}_{=0} \)
So:
\[R \left( \hat{\theta}_n \right) = Var\left(\hat{\theta}_n\right) + \text{bias}^2\left(\hat{\theta}_n\right)\]


Examples:
Say we have \(n\) random variable \(X_1,\cdots,X_n \sim \mathcal{U}([a,a+1])\)
What is \(\mathbb{E}\left[\overline{X}_n\right]\)
\(\mathbb{E}[X_i]=a+1/2\)
\(\mathbb{E}\left[\overline{X}_n\right]=\frac{\mathbb{E}[X_1]+\cdots +\mathbb{E}[X_n]}{n}\)
\(\mathbb{E}\left[\overline{X}_n\right]=\frac{1}{n}[\underbrace{(a+1/2)+\cdots +(a+1/2)}_{\text{n times}}]\)
\(\Rightarrow \mathbb{E}\left[\overline{X}_n\right]=a+1/2\)
Is \(\overline{X}_n - \frac{1}{2}\) an unbiased estimator for \(a\)
\(\mathbb{E}\left[\overline{X}_n -\frac{1}{2}\right]=\mathbb{E}\left[\overline{X}_n\right]-\frac{1}{2}\)
\(=(a+1/2)-1/2=a\)
So \(\mathbb{E}\left[\overline{X}_n - \frac{1}{2}\right]=a\Rightarrow\) that \(\overline{X}_n - \frac{1}{2}\) is an unbiased estimator for \(a\)
What is variance of \(\overline{X}_n-\frac{1}{2}\)
\(\text{var}\left(\overline{X}_n-\frac{1}{2}\right)=\text{var}\left(\overline{X}_n\right)\)
\(=\text{var}\left( \frac{X_1+\cdots +X_n}{n} \right) = \frac{1}{n^2}\left( \text{var}(X_1)+\cdots+\text{var}(X_n) \right) \)
  • \( \text{var}(X_i)=\frac{1}{12} \)
    because \( \text{var}\left(\mathcal{U}([a,b]) \right)=\frac{(b-a)^2}{12}\)
\(\Rightarrow \text{var}\left(\overline{X}_n-\frac{1}{2}\right) =\frac{1}{12n}\)
What is Quadratic Risk \(R\left( \overline{X}_n-\frac{1}{2} \right)\)
\( R \left( \hat{\theta}_n \right) = Var\left(\hat{\theta}_n\right) + \text{bias}^2\left(\hat{\theta}_n\right) \)
\(R\left( \overline{X}_n-\frac{1}{2} \right) = \frac{1}{12n}+0=\frac{1}{12n}\)
What is Quadratic risk telling us:
\(R\left( \hat{\theta}_n \right) := \mathbb{E} \left[ \left( \hat{\theta}_n - \theta \right)^2 \right] \)
So the quadratic risk is the expected squared distance between \(\hat{\theta}_n \) and \(\theta\) so we can take the square-root of the quadratic risk, and we can think of it as distance between \(\hat{\theta}_n \) and \(\theta\) and that is \(=\frac{1}{\sqrt{12n}} \xrightarrow [n\to \infty ]{} 0 \).
