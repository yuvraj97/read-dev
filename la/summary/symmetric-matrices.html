Symmetric Matrices

A \(n\times n\) matrix say \(A\) is symmetric if \(A=A^T\)

Properties
\(\bullet\)   Eigenvalues of a symmetric matrix are real
\(\bullet\)   For a symmetric matrix we can get orthonormal eigenvectors
\(\bullet\)   For a symmetric matrix signs of pivots are same as signs of eigenvalues
\(\bullet\)   If all the eigenvalues of a Symmetric matrix are positive then that matrix is a positive definite matrix
\(\bullet\)   If all the leading determinants of a Symmetric matrix are positive then that matrix is a positive definite matrix
